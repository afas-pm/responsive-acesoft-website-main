# Acesoft Mobile App Development Website

A modern, responsive website for Acesoft Technologies showcasing iOS and Android mobile application development services.

## Features

- **Responsive Design** - Optimized for all devices (mobile, tablet, desktop)
- **Modern UI/UX** - Clean, professional interface with smooth animations
- **Mobile-First Approach** - Designed for mobile devices first, then scaled up
- **Smooth Scrolling** - Seamless navigation between sections
- **ScrollReveal Animations** - Engaging animations as users scroll
- **Contact Form** - Functional contact form with validation
- **Interactive Navigation** - Active link highlighting and smooth transitions

## Pages

1. **Home** (`index.html`) - Main landing page with services overview
2. **iOS Development** (`ios.html`) - Dedicated iOS app development services
3. **Android Development** (`android.html`) - Dedicated Android app development services
4. **Contact** (`contact.html`) - Contact form and business information with embedded map

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with CSS variables
- **JavaScript** - Interactive functionality
- **ScrollReveal.js** - Scroll animations
- **Remix Icons** - Icon library
- **Google Fonts** - Inter font family

## Setup Instructions

1. **Clone or download** the project files
2. **Open** `index.html` in a web browser
3. **For local development**, use a local server:
   ```bash
   # Using Python
   python -m http.server 5500
   
   # Using Node.js (http-server)
   npx http-server
   ```

## Customization

### Colors
Edit CSS variables in `assets/css/styles.css`:
```css
:root {
  --red: #d32f2f;
  --black: #0f0f0f;
  --white: #ffffff;
  --gray: #6b7280;
  --light-gray: #f5f6f8;
}
```
